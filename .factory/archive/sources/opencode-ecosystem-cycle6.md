# OpenCode Ecosystem Update (May 2026)

OpenCode (150k+ GitHub stars) now has:

- **opencode-plugin-otel**: OTel exporter mirroring Claude Code's telemetry signals
- **Client/server architecture**: headless server + multiple frontends
- **Agent system**: plan/build modes
- **LSP integration**: code intelligence
- **Non-interactive scripting mode**

## Relevance to ccm

OpenCode and Claude Code are converging on similar architectures. ccm is one of the few tools that monitors both, reinforcing its agent-agnostic value proposition.
